git-novice
==========

An introduction to version control for novices using Git, based off the Software Carpentry curriculum at <https://swcarpentry.github.io/git-novice/>.
Please see <https://zuckermanbrain.github.io/git-novice/> for a rendered version of this material,
[the lesson template documentation][lesson-example]
for instructions on formatting, building, and submitting material,
or run `make` in this directory for a list of helpful commands.

This forked repository is maintained by the Zuckerman Institute Research Computing staff.
